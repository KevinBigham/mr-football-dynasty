# Commit Conventions

Use the slice id in commit subject for traceability.

## Format
`S###: <concise description>`

## Examples
- `S013: add check runner and grouped check utility`
- `S056: emit preload report json`
- `S079: publish workflow summary markdown`

## Batch Commits
If combining 5 slices:
`S021-S025: extract validation check modules (config/systems/data)`
